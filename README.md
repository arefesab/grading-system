A grading system provides the opportunity to define students and lessons and, to assign a score for each lesson to a student. It also allows creating, editing and deleting students and lessons.
The system identifies the students with the best average scores and also highlights the lessons with the highest number of failures. It can show you each student's lessons and scores as well as attendance and scores for each lesson.
The grading system was implemented with ASP.NET framework in Visual Studio Code, and the output is windows forms.For date storage, SQL Server is used.
There is a PDF and a film that show how this system works in Persian.
